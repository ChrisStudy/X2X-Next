## Overview

A modern analytics dashboard focused on performance tracking and data visualization. The UI is highly interactive and designed for clarity, flexibility, and responsiveness.

## Key Features

- Live-updating data visualizations
- Interactive charts powered by D3.js and Recharts
- Modular and customizable dashboard layout
- PDF export for reporting
- Smooth animations and transitions

## Tech Stack

Built with **React** and **Tailwind CSS**, enhanced by **Framer Motion** for animations and **D3.js** for advanced data visualization.
